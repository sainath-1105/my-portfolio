# Portfolio

A modern, high-impact personal portfolio with a sharp UI design, featuring bold typography and edgy animations.

## Features

- **Dynamic Hero Section**: Features a typing effect and a 3D background scene.
- **Video Integration**: Includes a custom video background in the hero section.
- **Animated Skills**: Skill icons with subtle, looping animations using Lottie and Framer Motion.
- **Responsive Design**: Fully responsive layout built with Tailwind CSS.
- **Custom Cursor**: Interactive custom cursor for a unique user experience.
- **Smooth Animations**: Powered by `motion/react` for a fluid feel.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (`motion/react`), Lottie (`react-lottie-player`)
- **3D**: Three.js, React Three Fiber
- **Icons**: Lucide React

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Environment Variables

- `GEMINI_API_KEY`: Required for any AI-powered features (if applicable).
